#vagex-setup
===========

vagex setup script for debian

vagex vision:1.4

#功能：
	vncserver挂vagex
	采用vncserver+iceweasel+vagex1.4
#备注：
 尽量在screen下运行以防中途断线
用法： 安装完成后用vnc连接后进行相应firefox优化后, File > Open File ，选择 Video Quality Manager 安装文件，安装后再安装 vagex_firefox_add_on*.xpi
 firefox优化无非就是关掉更新，关历史记录，更改首页为空白页面……

下载整个文件夹，在文件夹中
bash vagex-setup.sh
